Product: Template for WSO2 Siddhi-extension

STEPS:

 1.Navigate to "<siddhi-maven-archetype>/siddhi-extension-archetype" and run the following command
         mvn clean install
 2.After the successful build , Navigate to folder which is used to create your project

 3. Follow one of the steps below, which depends on your project

    * If you want to create a project as siddhi-execution
        Run the following command
            mvn archetype:generate
                -DarchetypeGroupId=org.wso2.siddhi.archetype.execution
                -DarchetypeArtifactId=siddhi-archetype-execution
                -DarchetypeVersion=1.0.0-SNAPSHOT
                -DgroupId=org.wso2.extension.siddhi.execution
                -Dversion=1.0.0-SNAPSHOT

        Enter the execution name
            After executing above command then the system will pop-up the following message
            "Define value for property 'executionType': "
                 eg:- Define value for property 'executionType': ML

        Finally Confirm the all property values
            If all property values are correct then type Y or click Enter, else type N
                 eg:- Y : : Y/click Enter


    * If you want to create a project as siddhi-io,
